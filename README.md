# lifetouch-cma-cli README

个人插件，用于 CMA 项目的开发

## Features

1. 将 js 语言包一键转为 json 语言包
2. svg-icon 组件插入图标预览
3. 左侧面板显示svg列表树，并且可以预览，复制代码
4. RouteManager提供路由查看树，更快定位路由文件和界面文件位置
5. Explorer面板提供vuex引用浏览定位功能，快速定位store文件
6. Explorer面板提供vue组件引用浏览定位功能，快速定位所引用的vue组件文件
